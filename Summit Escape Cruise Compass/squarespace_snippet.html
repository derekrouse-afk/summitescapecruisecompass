Squarespace: One Copyable Snippet

Click Copy and paste into a Squarespace Code Block. If Squarespace removes <script> tags, follow the instructions below.

If Squarespace strips <script> tags when pasting into a Code Block: paste the <style> block into Design → Custom CSS, paste the <script> block into Settings → Advanced → Code Injection → Footer, and paste only the HTML fragment into the page's Code Block.

After pasting, upload images to Squarespace and replace assets/images/logo.png with the hosted image URL.
